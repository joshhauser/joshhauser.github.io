Application mobile - Portes ouvertes de l'IUT de Vélizy

Cette application mobile est destinée à être utilisée par les personnes visitant l'IUT de Vélizy lors des portes ouvertes. Elle a pour objectif de présenter le DUT Informatique, l'IUT de Vélizy ainsi que la vie des étudiants (loisirs, restauration, logement...). Elle est particulièrement utile pour fournir facilement les informations les plus importantes, mais également pour susciter l'intérêt des futurs étudiants.

Support: Android
Ressources:

  • IDE Android Studio
  • Languages Java et XML
