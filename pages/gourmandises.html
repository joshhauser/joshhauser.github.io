LES GOURMANDISES

Ce projet est un site créé à l’aide d’un CMS (WordPress) dans le cas d’un mini-projet d’études en DUT Informatique. Le site présente les produits, services, et nouveautés de la boulangerie « Les Gourmandises », située à Dijon.
Il permet également aux clients de contacter la boulangerie ou d'envoyer des suggestions.

Note: le site étant hébergé sur un serveur privé (obligation imposée par l’IUT de Dijon-Auxerre), il n’est pas accessible en dehors du réseau interne de l'IUT.

Ressources:

  • CMS WordPress
  • Langage CSS
  • Base de données (SGBD phpMyAdmin)
  • Hébergeur: serveur interne à l'IUT
