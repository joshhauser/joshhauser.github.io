Cette application web est une plateforme permettant de gérer des listes de tâches, afin d'organiser son emploi du temps. L'utilisateur peut créer différentes listes pour séparer ses tâches en fonction de leur priorité, de leur type (tâches liées au travail, à la vie personnelle, liste de courses...).

Chaque liste possède un nom et peut également posséder une couleur.
Ces paramètres peuvent être édités par l'utilisateur.

Chaque liste ainsi que chaque tâche peut être supprimée lorsque l'utilisateur le souhaite.

Ressources:

  • Langages HTML5, CSS3, PHP 7 et SQL
  • Framework Angular 7 (framework écrit en TypeScript)
  • Serveur web Apache 2
  • Serveur MySQL
  • SGBD phpMyAdmin
  • NPM, gestionnaire de paquets Node.js
  • Angular Material: librairie de composants pour Angular
  • Material Icons: bibliothèques d'icônes
